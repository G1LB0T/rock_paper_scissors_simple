# Rock, Paper, Scissors — Version 1  
A simple two‑player Rock–Paper–Scissors game written in Python.  
This project was created as part of my programming learning journey and focuses on basic input handling, validation, conditional logic, and scoring.

---

## Features
- Two‑player gameplay  
- Player name input  
- Input validation for Rock, Paper, and Scissors  
- Clear explanation of game rules  
- Win/draw detection  
- Score tracking  
- Replay prompt  
- Beginner‑friendly, readable code

## Game Rules
Rock–Paper–Scissors follows three simple rules:

- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  

If both players choose the same option, the round ends in a draw.

---

## Program Flow
1. Program welcomes both players.  
2. Players enter their names.  
3. Rules are displayed.  
4. Player 1 chooses a move (validated).  
5. Player 2 chooses a move (validated).  
6. Winner is determined.  
7. Score is updated.  
8. Replay prompt appears (V1 ends after this).
